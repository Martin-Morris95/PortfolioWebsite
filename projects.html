Help Desk Notes Tool

[C#] [.Net] [Visual Studio] [WPF] [XAML] [SQL] [LINQ]

Features

  • Upload work related job notes to database
  • Word template generation for new notes to ensure harmonious design/style
  • Search functionality with tags and title
  • Caters for multiple users
  • Compiles all notes in one easy to access location

This was the first application I made upon starting my new job as an IT Support Officer. The idea came about when I was being trained alongside another employee. As you could expect each of us had many questions and we would write down the answers in our notebooks. This became messy quickly and made finding the answers to problems a lengthy process. We also began asking the same questions repeatedly as we both worked different days.

I realised we needed an efficient way to compile our knowledge, allow us to share notes, tips and tricks whilst storing them securely so the tool can be used by future new employees giving the company an easier way to train my role for future employees.

I created the application in Visual studio using C# and WPF. The application allows users to generate a note template (which opens word automatically), they then add searchable tags and a description and then upload it to the database. All the IT team can then open the app and search for that note.
